# Roadmapping for Success

**Elevate your stakeholder buy-in.**

A complete, reusable workshop kit from the hands-on session delivered at **Your Architect Dreamin' Helsinki 2026** (11 & 12 June 2026) by **Christopher Ramm** (Salesforce CTA, Salesforce MVP Class of 2025, DCX CTO Germany @ Capgemini) and **Sam Wadhwani** (Salesforce CTA, Salesforce MVP, Chief Technical Architect @ PwC).

> **Live workshop kit:** https://rammc.github.io/roadmapping-for-success/

## What is this?

A self-contained workshop kit for facilitators who want to run the "Roadmapping for Success" exercise with their own teams, communities, or user groups. Everything you need is in this repository: theory deck content, the Victory Fashion scenario, distractor cards, a beat-by-beat facilitator playbook, and four worked roadmap examples (including the cautionary `roadmap_final_v7_FINAL(2).pptx` anti-pattern).

The session teaches architects, product owners, and delivery leads how to build and evolve roadmaps that survive contact with reality – that absorb stakeholder change without collapsing, and that keep technical delivery traceable to business value.

## How to run the session

1. Read `materials/facilitator-playbook.md` end-to-end. It is the only thing you need on the day.
2. Print one copy of `materials/scenario-victory-fashion.md` per participant.
3. Print and cut the cards in `materials/distractor-cards.md` – one set per team.
4. Bring the four SVG examples up on screen during Block 1 (theory).
5. Run the session. Adapt timings as needed; the playbook flags every assumption.

Default session length: **90 minutes**. A 2-hour variant is documented in the playbook appendix.
